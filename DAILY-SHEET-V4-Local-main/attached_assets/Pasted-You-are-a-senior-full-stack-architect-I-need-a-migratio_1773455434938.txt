You are a senior full-stack architect. I need a migration/structure plan only (no product feature changes) to make this app easy to offload from hosted services to localhost/self-hosted environments.

## Current app context
- App: Daily Sheet (production scheduling + crew coordination).
- Frontend: Vite React app (currently deployed on Cloudflare Pages).
- Backend: Express API (currently deployed on Render).
- Database: PostgreSQL.
- Session auth is cookie-based with credentialed requests.
- Frontend env currently uses VITE_API_URL.
- Backend env currently includes DATABASE_URL, SESSION_SECRET, CORS_ORIGIN, optional RESEND_API_KEY and optional R2 storage vars.
- Build commands include:
  - npm run dev
  - npm run build
  - backend-only build via npx tsx script/build-server.ts

## What I want from you
Create a practical, step-by-step restructuring plan to run reliably on localhost and be production-portable later.

### Hard constraints
1) Do NOT write app feature code.
2) Do NOT redesign business logic.
3) Focus on architecture, environment separation, repo structure, and deployment workflow.
4) Keep auth/session behavior compatible.
